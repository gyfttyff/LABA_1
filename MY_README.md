# Rectangle Calculator

Программа для расчета параметров прямоугольника.

## Описание проекта

Данный проект представляет собой простой калькулятор на Python для вычисления:
- **Периметра** прямоугольника (P = 2 × (a + b))
- **Площади** прямоугольника (S = a × b)
- **Диагонали** прямоугольника (d = sqrt(a² + b²))

## Назначение

Программа предназначена для быстрого расчета геометрических параметров прямоугольника по заданным длинам сторон.

## Требования

- Pyton версии 3.1 и выше
- Модуль math (входит в стандартную библиотеку)
