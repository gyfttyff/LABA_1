# ЗАДАНИЕ 1: ПУНКТ b)
В программе реализуются функции вычисления параметров для треугольника по известным трём его сторонам. 
Таких как: Периметр, Площадь по формуле Герона, Проверка на равнобедренность 
# Функция test_of_exist 
Возвращает строку, проверяет через неравенства треугольников может ли сущетсвовать такой треугольник 
# Функция perimeter
Возвращает float в виде суммы всех сторон треугольника (т.е. его периметр)
# Функция square_geron
Возвращает переменную s, которая является площадью, а переменная popyper - полупериметром, которая используется для подсчёта s (площади) через формулу герона в этой функции
# Функция equisceles_check 
Возвращает строку YES или NO, определяя является треугольник равнобедренным (YES) или нет (NO).
# Функция main
Основная функция в которой выводятся все параметры и запускаются все функции.
